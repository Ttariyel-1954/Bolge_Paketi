# Test Faylları

Bu qovluqda admin panel tərəfindən yaradılmış test faylları (.rds) saxlanılır.

## Fayl formatı:
\`TEST_YYYYMMDD_HHMMSS.rds\`

Məsələn: \`TEST_20251110_065059.rds\`

## Mövcud testlər:
- TEST_PILOT_001.rds - Pilot test
- TEST_20251105_105415.rds
- TEST_20251106_200907.rds
- TEST_20251106_202057.rds
- TEST_20251107_074709.rds
- TEST_20251107_131526.rds
- TEST_20251110_065059.rds

## İstifadə:
Test app bu faylları oxuyur və şagirdlərə təqdim edir.
